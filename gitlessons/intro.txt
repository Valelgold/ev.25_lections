GIT - распределённная система контроля версий.Это система для отслеживания и ведения истории изменения файлов, в вашем проекте.

Репозиторий - хранилище вашего кода и истории его изменений 

GitHub - Веб сайт для размещения git - репозиториев в совместной разработке проектов.

Команды Git: 

1. Git init - она создает новый локальный гит репозиторий у вас на пк, в той директории в которой была прописана команда.(команда вводится один раз в начале)

2. Git add - команда для добавления файлов в систему отслеживания гита, после этой команды изменения начинают отслеживаться гитом.

3. Git commit -m '<comment>' - как только достигаем определенного момента в разработке, то тогда мы сохраняем и комментируем изменения в локальный репозиторий

4. Git remote add - команда для того, чтобы связать наш локальный репозиторий с удаленным репозиторием, то есть добавляет подключение к репозиторий на git_hub.
	git remote add <name: origin <ссылка на репо:ssh>

5. Git push - команда для отправки всех изменений на локальном репозитории в удачный репозиторий.
	git push origin <branch (main)>
