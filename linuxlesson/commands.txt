Linux commands 

Пишем в spot_land terminal - способ открытия терминала 
~ - символ домашней директории 
/ - символ корневой директории 
.. - обозначают предыдущие директорию 
. - обозначает текущую директорию 


1. Ls(list) - команда для вывода всех директорий(папка) и файлов в текущей директории в терминале.

2. Cd (change directory - команда для изменения(переходов) одной директории на другую.
        Cd <название директории>
   
3. Mkdir (make directory) - Создает указанную директорию в рабочей директории в терминале.

4. Touch - Создает указанный файл.

5. Rm (remove) - удаление файла.

6. Rmdir - удаление директории (если есть файл, то удаляться не будет)

7. Pwd (Present working directory) - выводит путь до текущей директории.

8. rm -rf - команда принудительного удаления.

9. Ls -a - команда, которая выводит список как скрытых так и обычных файлов директорий.

10. Nano <path_to_file> - откроется файл для редактирования в графическом интерфейсе.
*если нет файла, то  nano попытаеться создать новый файл

11. Cat <file_name> - выводит содержимое файла в терминал.

12. Echo - комнанда для быстрой записи текстовой информации в какой либо файл. 
    Echo (текст для записи) > Название файла

13. Mv - команда для переименования файла, либо директории, а также для перемещения файлов или директории.
   Переименовывание: mv jonh john 
                     Mv <старое название> <новое название>
   Перемещение: mv John ~/Desktop/
                Mv <file/directory> <directory>/
